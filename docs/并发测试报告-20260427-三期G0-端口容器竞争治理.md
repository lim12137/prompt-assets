# 并发测试报告：三期 G0 端口/容器竞争治理（2026-04-27）

## 背景

- 目标脚本：`scripts/run-admin-category-management-real-db-e2e.mjs`

## 改动摘要

- 引入全程锁：接入 `withTestDbLock`，主流程放入锁内执行。
- 独立端口：新增默认 `TEST_DB_PORT=55434`，并以该端口构造默认 `TEST_DATABASE_URL`。

- 独立前端资源：新增默认 `PLAYWRIGHT_WEB_PORT=3113`、`PLAYWRIGHT_WEB_DIST=.next-e2e-real-db-admin-category`。
